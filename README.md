# Fandom Harry Potter

## O Projeto: [Fandom Harry Potter](#)

## Foi feito com 
- HTML5 Semântico
- CSS Colors e Fonts(font-face)
- CSS Flebox
- CSS Grid
- Design Responsivo(CSS Media Queries)

## O que foi aprendido

No HTML, dou destaque para os acordeões da página 'A Saga' com as tags _details_, _summary_ e _div_ para exibir a imagem e um resumo de cada livro.

No CSS, o desafio foi a estilização do cabeçalho e menu principal do site, adicionando uma imagem de fundo ao título principal e a adaptação do menu principal para desktops(menu horizontal) e também para dispositivos mobile(menu vertical).

## O uso da IA

Foi utilizado o ChatGPT para a criação das imagens e de parte dos textos.
Nas imagens, solicitou-se a criação das imagens no estilo do _studio Ghibli_.Os prompts foram específicos - definindo como deveria ser o plano de fundo da imagem, as características marcantes dos personagens e as feições de acordo com os livros.
Quanto aos textos, foi pedido apenas para produzir diferentes tipos de texto:
- fazer um resumo de cada um dos personagens citados, destacando suas características principais
- organizar as informações e curiosidades da história em listas
